# TextTool

The TextTool behavior is present on 3DText objects i.e those created with the [text tool](https://kb.tabletopsimulator.com/game-tools/text-tool/).

## Functions {#functions}

|Function Name|Return|Description|
|---|---|---|
|getFontColor()|return `color`|Returns Table of font Color.|
|getFontSize()|return ` int `|Returns Int of the font size.|
|getValue()|return ` string `|Returns the current text. Behaves the same as Object's [getValue()](../../object/#getvalue).|
|setFontColor( ` color ` font_color)|return ` boolean `|Sets font Color.|
|setFontSize( ` int ` font_size)|return ` boolean `|Sets font size.|
|setValue( ` string ` text)|return `boolean`|Sets the current text. Behaves the same as Object's [setValue(...)](../../object/#setvalue).|
