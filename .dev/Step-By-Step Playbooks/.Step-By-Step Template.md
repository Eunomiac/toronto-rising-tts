## Three layers (do not conflate)

| Layer | Name | Drives splits? |
| --- | --- | --- |
| Author run order | **Step 1, Step 2, …** | Paste IDE block or bold manual click |
| Paste boundaries | **Code Block 0, A.1, A.2, …** | Split only at human gates between pastes |
| Console structure | **Phase 1.1, 2.1, …** | Never — `printHeader(..., 1)` labels inside a block |

Subjective HUMAN lines (lighting, timing, layout eyeball) are a last resort when Lua assert would be prohibitively complex.

## Code Block 0 — Prerequisites

```lua
U.RunSequence({
  function()
    printHeader("Title: Prerequisites", 1)
  end,
  function()
    local lines = {}
    local function check(label, ok, detail)
      local suffix = ok and "PASS" or ("FAIL — " .. tostring(detail))
      table.insert(lines, ">> " .. label .. ": " .. suffix)
      return ok
    end
    local allOk = true
    local seated = Player.getPlayers() or {}
    local numClients = #seated
    allOk = check("Host connected (solo OK)", numClients >= 1, "no players seated") and allOk
    allOk = check("Single client (typical solo dev)", numClients <= 1, numClients .. " connected") and allOk
    allOk = check("Storyteller seat (Black)", U.isStorytellerSteamPlayer("Black"), "assign Black in setup") and allOk
    local tableKey = S.getStateVal("seatLayout", "currentTableKey")
    allOk = check("Current table is Table A", tableKey == "Table A", "got " .. tostring(tableKey)) and allOk
    print(table.concat(lines, "\n"))
    if not allOk then
      error("[Prereq FAIL] fix prerequisites before continuing")
    end
  end,
  function()
    printHeader("", 1)
  end,
  function()
    print("   ▶▶▶ HUMAN ▶▶▶ Run Code Block A.1.")
  end,
})
```
