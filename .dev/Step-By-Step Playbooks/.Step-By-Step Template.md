## Three layers (do not conflate)

| Layer | Name | Drives splits? |
| --- | --- | --- |
| Author run order | **Step 1, Step 2, …** | Paste IDE block or bold manual click |
| Paste boundaries | **Code Block 0, A.1, A.2, …** | Split only at human gates **(1)–(4)** ([skill](../../.cursor/skills/step-by-step-guidance/SKILL.md#human-gates-when-to-stop-automation)) |
| Console structure | **Phase 1.1, 2.1, …** | Never — `printHeader(..., 1)` labels inside a block |

Subjective HUMAN lines are a last resort (human gate **(2)**) when Lua assert would be prohibitively complex. Simple drops/clicks/rolls are gate **(1)** — do not simulate them in Lua.

## Code Block 0 — Session setup + verify

Pattern: **prepare** → **verify** → more automated phases in the **same** sequence until a human gate or completion.

```lua
-- Optional: hoist test constants for later blocks (session stays loaded).
local FIXTURE = {
  tableKey = "Table A",
  sceneSlotIndex = 3,
  sceneKey = "e2e_verify_stub",
  npcA = "myleneHamelin",
}

U.RunSequence({
  function()
    printHeader("Title: Session setup", 1)
  end,
  function()
    if #(Player.getPlayers() or {}) < 1 then
      error("[Setup FAIL] Host not connected")
    end
    rollE2eSeatPrep("Black")
    DEBUG.syncTableSimplified(FIXTURE.tableKey)
    if type(DEBUG.ensureSceneLibraryStub) ~= "function" then
      error("[Setup FAIL] DEBUG.ensureSceneLibraryStub missing — Save & Play after pulling latest repo Lua")
    end
    DEBUG.ensureSceneLibraryStub(FIXTURE.sceneSlotIndex, FIXTURE.sceneKey, {
      title = "E2E verify stub",
      tableKey = FIXTURE.tableKey,
    })
    DEBUG.spawnNpcControlBoardTokens({ destroyExisting = false })
    print("PASS — session prepared (seat, table, tokens, library stub)")
  end,
  function()
    printHeader("Title: Verify ready", 1)
  end,
  function()
    if not U.isStorytellerSteamPlayer("Black") then
      error("[Verify FAIL] Host not on Black after rollE2eSeatPrep")
    end
    if S.getStateVal("seatLayout", "currentTableKey") ~= FIXTURE.tableKey then
      error("[Verify FAIL] table key not " .. FIXTURE.tableKey)
    end
    print("PASS — prerequisites satisfied")
  end,
  -- Merge automated steps from "Code Block A" below into THIS sequence when no human gate separates them.
  -- Do NOT end with handoff HUMAN ("Run Code Block …") — that is not a human gate.
})
```

## Code Block A — Automated body (merge into Code Block 0 when Lua-only)

**Agent note:** This fence is separate in the template for readability. In production playbooks, **paste it as continuation steps in the same `U.RunSequence`** as Code Block 0 unless a human gate **(1)–(3)** falls between.

```lua
local TITLE = {}

-- ILLUSTRATION ONLY — use rollConfirm / domain harness in real playbooks (see README helper registry).
local function dummyFunction_X(param1, param2)
  return true, "ok"
end

U.RunSequence({
  function()
    printHeader("PHASE 1.1 — Initializing illustration helpers", 1)
    TITLE.helperFunctionOne = function()
      return true
    end
    TITLE.helperFunctionTwo = function()
      return true
    end
    TITLE.helperFunctionThree = function()
      return true
    end
  end,
  function()
    print("Helper functions initialized.")
  end,
  function()
    printHeader("PHASE 2.1 — First test of dummyFunction_X", 1)
  end,
  function()
    local param1, param2 = "alpha", 42
    print(">> Calling dummyFunction_X(" .. tostring(param1) .. ", " .. tostring(param2) .. ") ...")
  end,
  function()
    local param1, param2 = "alpha", 42
    local logLines = {}
    local ok, result = xpcall(
      function()
        return dummyFunction_X(param1, param2)
      end,
      function(err)
        return tostring(err) .. "\n" .. debug.traceback("", 2)
      end
    )
    if not ok then
      table.insert(logLines, "")
      table.insert(logLines, "FAIL — dummyFunction_X raised an error")
      table.insert(logLines, "   " .. tostring(result))
      table.insert(logLines, "")
      print(table.concat(logLines, "\n"))
      error("[Phase 2.1 FAIL] dummyFunction_X")
    end
    -- Real playbooks: prefer rollConfirm({ ... }) or domain DEBUG asserts instead of custom PASS lines.
    table.insert(logLines, "PASS — dummyFunction_X returned " .. tostring(result))
    print(table.concat(logLines, "\n"))
  end,
  function()
    -- Human gate (2) or (3): subjective visual — split to Code Block B for post-human asserts.
    print("   ▶▶▶ HUMAN ▶▶▶ Visually confirm the Purple player's hand zone is correctly positioned and rotated, then run Code Block B.")
  end,
})
```
