Compact Syntax Guide

  • Regex patterns use JS literal text: /pattern/giu; replacements may use $1.
  • Ranges accept A1/named ranges, including quoted sheets: 'Sheet Name'!A1:D.
  • Markdown source requires path, tags, and optional third yaml column.
  • Links support [[Page]], [[Page#Heading]], and [[Page#Heading|Visible]].
  • Tags are YAML frontmatter, lower camelcase, alphanumeric plus underscore.
