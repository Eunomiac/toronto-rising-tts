# Counter

The Counter behavior is present on the Counter object.

## Functions {#functions}

|Function Name|Return|Description|
|---|---|---|
|clear()|return `boolean`|Resets Counter to 0.|
|decrement()|return ` boolean `|Reduces Counter's value by 1.|
|getValue()|return ` int `|Returns Counter's current value. This function behaves the same as [Object's getValue()](../../object/#getvalue).|
|increment()|return ` boolean `|Increases Counter's value by 1.|
|setValue()|return ` boolean `|Sets the current value of the Counter. This function behaves the same as [Object's setValue()](../../object/#setvalue).|

> **Example: Example**
> Increment a counter's value.
>
> ```lua
> object.Counter.increment()
> ```
>
> ```lua
> object.Counter.increment()
> ```
