```markdown
Read this when:
- touching `<path-or-system>`
- changing `<behavior>`

Do not read this when:
- working on `<unrelated-system>`

Source of truth:
- `<code-or-data-path>`

Generated/dependent files:
- `<generated-path>` - regenerate with `<command>`

Verification:
- `<command-or-manual-playbook>`

Status:
- current | needs verification | historical | generated
```
